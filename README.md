Description Print Lab is a PHP and MySQL based web application that allows users to browse a curated collection of photography prints, add items to a shopping cart, update quantities, and proceed to checkout. The application focuses on clean MVC and UI design, structured server-side logic, and foundational e-commerce functionality.

Project Features

Responsive navigation and layout
Product listing from a MySQL database
Shopping cart functionality (add, update, remove items)
Quantity limits enforced per product
Cart total calculations
Checkout flow
MVC style folder organization
Technologies Used PHP, MySQL, HTML5, CSS3, XAMPP, phpMyAdmin, Git, GitHub

Setup Instructions

Install XAMPP
Start Apache and MySQL
Import SQL script into phpMyAdmin
Update database credentials
Place project in htdocs
Access via localhost
Testing Manual testing was performed on navigation, cart functionality, calculations, and checkout behavior.

Lessons Learned

Project planning is critical to smooth iteration.
Clean code and structure improve maintainability.
Clear naming conventions reduce confusion.
Documentation and comments support efficient development.
Version control discipline protects progress.
Future Improvements User authentication, persistent carts, order history, payment integration and improved styling.
